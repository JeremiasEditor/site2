# Assets Folder

Place the following files in this directory:

## Required Files

| File | Description |
|------|-------------|
| `reel.mp4` | Hero section video background — your best work reel |
| `texture.png` | Texture overlay image for the hero section aesthetic |
| `carlos-santana.jpg` | Profile/channel image for Carlos Santana project card |
| `viaja-britto.jpg` | Profile/channel image for Viaja Britto project card |
| `perfil-geek.jpg` | Profile/channel image for O Meu Perfil Geek project card |

## Notes

- `reel.mp4` should be compressed for web (H.264, ~5–15 MB recommended for fast loading)
- `texture.png` can be a grain/noise texture, paper texture, or film grain overlay
- Profile images should be square or close to square (recommended: 400×400px minimum)
- All images should be optimized for web (use tools like Squoosh or TinyPNG)

## Optional

- `icon.svg` — Jeremias & co. brand icon (can replace the Clapperboard icon in the UI)
